# BeamLayoutAnalyzer

AutoCAD 2026 用の梁伏図（フレーミングプラン）構造解析プラグインです。  
AutoCAD上のオブジェクトからスラブ・柱・梁を自動認識し、負担面積と積載荷重を算出します。  
さらに、**負担面積図をモデル空間に自動作図**する機能を備えています。

## 機能

### 🏗️ ジオメトリ自動認識
- **閉じたポリライン** → スラブとして認識
- **円** → 柱として認識
- **線分** → 梁として認識

### 📐 大梁・小梁の自動判定
- 両端が柱に接続している梁 → **大梁**
- それ以外 → **小梁**

### 📊 負担面積の計算
- **梁**: 二等分線分割法（Bisector Partitioning Method）による支配面積算出
  - 全節点（柱・梁交差点）から隣接する梁の角度二等分線を生成
  - 反復トリミングで分割線の終点を正確に決定
  - 平面グラフの面列挙でサブパネルを検出し、各梁に面積を割当
- **柱**: ボロノイ分割（Voronoi Diagram）による支配面積算出

### 📏 通り芯の自動検出
- AutoCAD図面上の梁配置から通り芯（Grid Line）を自動検出
- 垂直・水平方向の通り芯を識別し、符号（X1, Y1 等）を自動付与
- スラブ境界線も通り芯として管理（符号なし）

### 🖨️ 負担面積図の自動作図
選択した大梁の負担面積図をモデル空間に自動生成します：

| 要素 | 説明 |
|------|------|
| **通り芯** | 名前付き通り芯（符号・円ラベル付き）+ スラブ境界通り芯 |
| **パネル寸法** | 名前付き通り芯間のセグメント寸法 + 全体寸法 |
| **スラブ境界寸法** | 名前付き通り芯～スラブ境界の距離（重複なし） |
| **負担面積ハッチング** | ANSI31 パターンで負担面積領域を表示 |
| **台形寸法** | 上底の幅 + 下底から上底までの高さ（長方形はスキップ） |
| **凡例** | ハッチングサンプル付きの「…負担面積」ラベル |

#### 作図ルール
- 全エンティティは `ByLayer` で生成
- 画層: `23_通り芯`, `22_文字`, `21_寸法線`, `01_RMN_新設設備`
- 寸法スタイル: `1-100`
- 通り芯の交点からの突き出し: 300mm
- 台形高さ寸法: 通り芯寸法の反対側（右/上）に配置、端点から800mm外側

### 🖥️ WPF ビジュアルビューア
3つの表示タブを切り替えて解析結果を確認できます：

| タブ | 内容 |
|------|------|
| **地震力** | スラブ面積 × 地震力定数の荷重表示 |
| **大梁・小梁** | 梁ごとの分割領域と積載荷重表示 |
| **柱** | 柱ごとのボロノイ領域と積載荷重表示 |

- 各部材をクリックすると、詳細情報（長さ、座標、支配面積、荷重）を右パネルに表示
- 積載荷重定数（kN/m²）はユーザーが自由に変更・再計算可能

## 使い方

### 1. ビルド

Visual Studio 2022 でソリューションを開き、両プロジェクトをビルドします。

```
ビルド出力: ..\BeamLayoutOutput\
  ├── BeamLayoutLoader.dll      ← AutoCAD にロードするDLL
  └── BeamLayoutAnalyzer.dll    ← 本体（シャドウコピーで動的ロード）
```

> **注意**: AutoCAD 2026 の DLL 参照パスがデフォルトで `C:\Program Files\Autodesk\AutoCAD 2026\` に設定されています。環境に合わせて `.csproj` を修正してください。

### 2. AutoCAD で読み込み

AutoCAD 2026 のコマンドラインで：

```
NETLOAD
```

→ **`BeamLayoutLoader.dll`** を選択して読み込みます（BeamLayoutAnalyzer.dll ではありません）。

### 3. コマンド実行

```
FRAMINGPLAN
```

### 3.5 コードを修正した場合（シャドウコピー再ロード）

コード修正後、Visual Studio でリビルドしたら：

```
RELOADBEAM
```

→ AutoCADを再起動せずに最新のDLLが反映されます。その後 `FRAMINGPLAN` を再実行してください。

> **ヒント**: `BEAMINFO` コマンドで現在のローダー・本体DLLの情報を確認できます。

### 4. オブジェクト選択

図面上のスラブ（閉じたポリライン）、柱（円）、梁（線分）を選択して **Enter** を押します。  
全選択する場合は **Ctrl+A** → **Enter** でも可能です。

### 5. ビューアで解析

WPF ウィンドウが開き、以下の操作が可能です：

- タブ切替で表示モードを変更
- 部材をクリックして詳細情報を確認
- 積載荷重定数を変更して「再計算・更新」ボタンで再計算
- **大梁をクリック → 「作図」ボタン** で負担面積図をモデル空間に自動生成

|------|-------------|

## 計算手法

### 二等分線分割法（梁の負担面積）

梁の負担面積は以下のステップで計算されます：

#### Step 1: ベース線分の収集
スラブ境界辺と梁の線分を収集します。浮き梁（片端が柱に未接続の梁）はスラブ境界まで延長されます。

#### Step 2-3: 二等分線レイの生成（Phase 1）
全節点（柱中心・梁交差点・梁端点）を走査し、各節点に接続するエッジ間の角度二等分線を生成します。

- 各節点で接続線分の方向角を収集
- 隣接するエッジ角度の二等分方向にレイを生成
- 360°/0°のラップアラウンドを考慮した重複除去

#### Phase 2: 分割線の終点決定（反復トリミング）

1. **Step A**: 全レイをスラブ境界まで延伸（fullLines）
2. **Step B&C**: 反復トリミング（最大5パスで収束）
   - 各fullLineに対し、梁・他の分割線との最近交差点を検出
   - 相手の分割線が実際に到達しない交差は除去（反復で収束）

#### Step 3.5: スキューネスポイント接続
ベース線分上にない分割線の自由端を検出し、同じパネル内の自由端同士を最近傍チェーンで接続します。

#### Step 4: 面列挙
全線分（ベース＋分割線）を平面グラフとして構築し、面を列挙してサブパネルを検出します。

#### Step 5: 面積割当
各サブパネルの辺を共有する梁に面積を按分します。

### ボロノイ分割（柱の負担面積）

各柱について、隣接する柱との垂直二等分線で Sutherland-Hodgman アルゴリズムを用いて半平面クリッピングを行い、スラブ内のボロノイセルを算出します。

## 座標系

AutoCAD 上の座標は **mm 単位** を想定しており、内部処理では自動的に **m 単位** に変換されます（1/1000 スケール）。
